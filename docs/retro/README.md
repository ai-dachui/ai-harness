# Experience Logs (Retro)

Session-by-session lessons captured by `/retro`. Monthly distillation by `/distill` extracts patterns into runbooks.

## Format

Each month gets one file: `YYYY-MM.md`

Entry format:
```markdown
### YYYY-MM-DD: Title

**Discovery/Pitfall/Decision**: Description including the "why"

[distilled→runbook/xxx.md] ← added by /distill when extracted
```

## Rules
- Only record new knowledge and real lessons — no play-by-play
- Every entry must include "why" (not just "what happened")
- /distill marks extracted entries, never deletes originals
